Basic HTML Cheatsheet

Basic tags

Tag Name Description Example
<html> Html Creates an HTML document 
                    
                        <html>
                            <head>
                                //page information 
                                <title>Youtube</title>      
                            </head>
                            <body>
                                //HTML code
                            </body>
                        </html>
<head> Head Sets off the title & other info that isn't displayed.
<body> Body Sets off the visible portion of the document.
<title> Title Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked.

List Tags

Tag Name Description Example Result
<ol> Ordered list

This defines an ordered list. An ordered list can be numerical or alphabetical

                 <ol>
                     <li>Clean room</li>
                     <li>Take out Garbage</li>
                     <li>Feed cats</li>
                 </ol>

<ul> Ordered list

This defines an unordered list. groups a collection of items having no numerical order

This element is used to represent an item in a list
